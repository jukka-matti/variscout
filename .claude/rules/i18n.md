# i18n — non-negotiables

- **No string concatenation** for translated content — use `formatMessage()` with parameters; Intl APIs do plurals/numbers/dates.
- **Apps call `registerLocaleLoaders()` at startup** before any `preloadLocale()`. Tests register their own loader via `import.meta.glob` before `beforeAll` — without it, all locales silently fall back to English.
- **Stat values use `formatStatistic()`** at the display boundary — never `.toFixed()` (ESLint enforced).

Detailed patterns + Chinese filename map + adding-message recipe: `packages/core/CLAUDE.md`.
