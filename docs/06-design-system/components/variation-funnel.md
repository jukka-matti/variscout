# Variation Bar (Removed)

> **ADR-062**: The `VariationBar` component and `cumulativeVariationPct` field were removed in the ADR-062 refactor. Filter chips now display `n=X` (sample count) instead of a Total SS contribution percentage. The variation funnel concept is still valid as a mental model, but the progress bar UI and cumulative tracking are no longer part of the product.

For the current filter chip design, see the [Filter Chips](../../03-features/navigation/breadcrumbs.md) documentation and the [Navigation Patterns](../patterns/navigation.md) page.

For the drill-down mental model, see [Drill-Down: Progressive Variation Analysis](../../01-vision/four-lenses/drilldown.md).
