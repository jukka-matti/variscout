## Production Flow

### Phase 1: Record (Batch)

Record 4 videos in 1-2 sessions (one month of content).

**Setup:**

- Good lighting
- Clean background
- Quality microphone
- Screen recording ready (for demos)

**Per video:**

- 5-10 minutes talking
- Use brief as guide (talking points, not script)
- Include VaRiScout demo moments where relevant

### Phase 2: Edit

**Full video (YouTube):**

- Light editing (cuts, remove mistakes)
- Add intro/outro
- Add captions
- Add chapter markers

**Clips (TikTok/Instagram):**

- Identify 3-5 strong moments
- Cut to 30-60 seconds each
- Vertical format (9:16)
- Add captions (essential for social)
- Add hook text overlay

### Phase 3: Transcribe & Write

**Transcript:**

- Auto-transcribe (YouTube, Descript, or similar)
- Clean up for readability

**Blog post:**

- Expand on transcript
- Add structure (headers, sections)
- Add images/screenshots
- SEO optimize (keywords, meta)

### Phase 4: Create Supporting Content

**LinkedIn posts:**

- Extract 2-3 key insights
- Write as standalone posts
- Link to full video/blog

**Instagram carousel:**

- 5-7 slides
- Key points visualized
- Consistent design template

### Phase 5: Schedule

Spread content across the week:

| Day           | Content                                  |
| ------------- | ---------------------------------------- |
| **Monday**    | YouTube video + Blog post                |
| **Tuesday**   | LinkedIn post #1 + TikTok clip #1        |
| **Wednesday** | Instagram carousel + TikTok clip #2      |
| **Thursday**  | LinkedIn post #2 + TikTok clip #3        |
| **Friday**    | LinkedIn post #3 (engagement/discussion) |
| **Weekend**   | Additional clips if available            |

## Tools

### Recording

| Tool                 | Purpose              | Notes                              |
| -------------------- | -------------------- | ---------------------------------- |
| **Camera**           | Phone or webcam      | Good enough quality                |
| **Microphone**       | Audio quality        | Essential - bad audio kills videos |
| **Lighting**         | Ring light or window | Consistent, flattering             |
| **Screen recording** | OBS / Loom / native  | For VaRiScout demos                |

### Editing

| Tool              | Purpose                  | Notes                          |
| ----------------- | ------------------------ | ------------------------------ |
| **Video editing** | DaVinci Resolve / CapCut | Free options available         |
| **Clip cutting**  | CapCut / Opus Clip       | Can auto-identify good moments |

| **Thumbnails**    | Canva                    | Consistent template            |

### Writing

| Tool              | Purpose                 | Notes                              |
| ----------------- | ----------------------- | ---------------------------------- |
| **Transcription** | YouTube auto / Descript | Start point for blog               |

## Semi-Automation Opportunities

### Clip Identification

- **Opus Clip** or similar can auto-identify "viral" moments
- Review and select best 3-5
- Manual final cut for quality

### Transcription → Blog

- Auto-transcribe video
- Use Claude to expand into blog post structure
- Manual review and polish

### Caption Generation

- Auto-generate captions
- Review for accuracy
- Style consistently

### Cross-Posting

- TikTok → Instagram Reels (same content)
- Clips can go to LinkedIn video (square format)

## Quality Checklist

### Video Recording

- [ ] Good audio (no echo, clear voice)
- [ ] Good lighting (face visible)
- [ ] Clean background
- [ ] Talking points covered
- [ ] Demo recorded (if applicable)

### Full Video (YouTube)

- [ ] Edited (cuts, pacing)
- [ ] Captions added
- [ ] Chapters marked
- [ ] Thumbnail created
- [ ] Title and description written
- [ ] Tags added

### Clips (TikTok/Instagram)

- [ ] Strong hook in first 2 seconds
- [ ] Captions added
- [ ] Vertical format (9:16)
- [ ] Under 60 seconds
- [ ] Call to action (subtle)

### Blog Post

- [ ] Clear structure
- [ ] Key points from video
- [ ] Images/screenshots added
- [ ] SEO keywords included
- [ ] Links to video and VaRiScout

### LinkedIn Posts

- [ ] Standalone value (don't require watching video)
- [ ] Hook in first line
- [ ] Clear insight or takeaway
- [ ] Link to full content

## Notes

- **Quality over quantity:** Better to post less but better content
- **Consistency matters:** Same day/time builds audience expectation
- **Batch when possible:** Recording 4 videos at once is more efficient
- **Repurpose everything:** One video = many pieces of content
- **Captions are essential:** Most social video is watched without sound
