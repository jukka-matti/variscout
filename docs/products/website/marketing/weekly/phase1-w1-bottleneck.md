### Post 2: The Insight (Wednesday)

```
John Tukey invented the box plot in 1977.

He also wrote: "Exploratory data analysis is detective work."

The ASQ Green Belt Body of Knowledge still tests stem-and-leaf plots.

(When did you last make one by hand?)

We kept Tukey's technique.
We missed Tukey's philosophy.

He wanted us to LOOK at data before testing it.
To explore before confirming.
To see patterns before calculating p-values.

Traditional Six Sigma inverted this.

"Which test should I run?" comes before "What does this look like?"

What if we went back to what Tukey actually meant?

#EDA #DataVisualization #SixSigma
```
