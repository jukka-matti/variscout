## Platform-Specific Notes

### YouTube

- Publish Monday morning
- Optimize title, description, tags
- Add chapters
- Add end screen (subscribe + next video)

### LinkedIn

- Monday: Video link + key insight
- Wednesday: Standalone insight post
- Friday: Discussion/question post
- Use personal profile (not company page)

### TikTok (@variationscouting)

- 3-5 clips per week
- Under 60 seconds each
- Strong hook in first 2 seconds
- Captions always
- Trending sounds (when relevant)

### Instagram (@variationscouting)

- Reels: Same as TikTok clips
- Carousel: Mid-week, key visual points
- Stories: Behind the scenes, polls (optional)

### Blog

- Publish with YouTube video
- SEO optimize for LSS keywords
- Link to VaRiScout Lite
- Include video embed
