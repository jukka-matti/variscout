## Voice & Tone

### Voice Attributes

| Attribute        | Description                | Example                                          |
| ---------------- | -------------------------- | ------------------------------------------------ |
| **Clear**        | No jargon unless necessary | "Different? YES" not "statistically significant" |
| **Confident**    | We know this works         | "Find the factor" not "might help you find"      |
| **Approachable** | Not intimidating           | "In minutes" not "rapid time-to-insight"         |
| **Practical**    | Focus on outcomes          | "Copy to PowerPoint" not "export capabilities"   |

### Tone by Context

| Context        | Tone                    | Example                                          |
| -------------- | ----------------------- | ------------------------------------------------ |
| Homepage       | Energetic, inviting     | "Start scouting variation now"                   |
| Product pages  | Informative, helpful    | "Here's what VaRiScout Web includes"             |
| Pricing        | Straightforward, fair   | "Free forever with watermark"                    |
| Documentation  | Clear, supportive       | "Select your outcome variable first"             |
| Error messages | Calm, solution-oriented | "No data selected. Choose a column to continue." |

## Words to Use

| Instead of...             | Use...                       |
| ------------------------- | ---------------------------- |
| Statistically significant | Different (with confidence)  |
| Null hypothesis           | Baseline expectation         |
| P-value (alone)           | "Different? YES (p=0.003)"   |
| Utilize                   | Use                          |
| Leverage                  | Use                          |
| Solution                  | Tool                         |
| Empower                   | Help                         |
| Best-in-class             | Simple / Fast / Clear        |
| Robust                    | Reliable                     |
| Seamless                  | Easy                         |
| Enterprise-grade          | Secure / Microsoft certified |

## Call-to-Action Patterns

### Primary CTAs

| Context       | CTA                      | Notes            |
| ------------- | ------------------------ | ---------------- |
| Homepage hero | "Try Free"               | Direct to /app   |
| Product pages | "Get Started"            | Product-specific |
| Pricing       | "Start Free" / "Buy Now" | Tier-specific    |

### Secondary CTAs

| Context          | CTA                | Notes            |
| ---------------- | ------------------ | ---------------- |
| Homepage hero    | "Watch Demo"       | Video or tour    |
| Feature sections | "Learn More"       | Deeper page      |
| Comparison       | "Compare Products" | /product/compare |

### CTA Phrasing Rules

- Start with verb
- Keep to 2-3 words
- Be specific about outcome
- Avoid "Submit" or "Click Here"

| ❌ Avoid           | ✅ Use      |
| ------------------ | ----------- |
| Submit             | Send        |
| Click here         | Learn more  |
| Get access         | Try free    |
| Start your journey | Get started |

## Messaging Don'ts

| Don't                             | Why                               |
| --------------------------------- | --------------------------------- |
| Claim to replace Minitab entirely | We're complementary for basic use |
| Promise AI/ML capabilities        | We're deterministic stats         |

| Make compliance claims            | We're not legal/compliance tools  |
