## Video Content Types

### Type A: Conversion Video (Week 1, 5, 9, 12)

- **Length:** 5-10 minutes
- **Structure:** Story → Problem → VaRiScout Reveal → CTA
- **Goal:** Drive to try VaRiScout
- **Example:** "The Bottleneck" - full case walkthrough

### Type B: Teaching Video (Week 2, 3, 6, 7, 10, 11)

- **Length:** 2-3 minutes
- **Structure:** One concept → Demo → Link to tool/learn page
- **Goal:** Quick win, practical skill
- **Example:** "How to read an I-Chart in 60 seconds"

### Type C: AI Comparison (Week 4, 8, 12)

- **Length:** 5-7 minutes
- **Structure:** Same data → VaRiScout analysis → Copilot Analyst → Compare
- **Goal:** Position VaRiScout value vs AI tools
- **Example:** "Can AI find the bottleneck?"
