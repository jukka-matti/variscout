# Power BI Custom Visuals

VariScout charts as Power BI custom visuals.

---

## Status

**Planned** - Not yet in development.

---

## Concept

Power BI custom visuals would bring VariScout charts to Power BI dashboards:

- Native Power BI data binding
- Linked filtering via Power BI slicers
- VariScout methodology in BI context

---

## Planned Visuals

| Visual     | Description               | Priority |
| ---------- | ------------------------- | -------- |
| I-Chart    | Control chart with limits | High     |
| Boxplot    | Factor comparison         | High     |
| Capability | Histogram with specs      | Medium   |
| Pareto     | Category ranking          | Medium   |

---

## Technical Approach

- Use Power BI visuals SDK
- Port Visx charts to visuals framework
- Share calculation logic from @variscout/core

## Distribution

- Power BI AppSource
