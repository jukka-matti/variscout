# @variscout/data

Sample datasets with pre-computed statistics and chart data for VariScout.

## Installation

```json
{
  "dependencies": {
    "@variscout/data": "workspace:*"
  }
}
```

## Exports

| Export Path                | Description                |
| -------------------------- | -------------------------- |
| `@variscout/data`          | Main entry                 |
| `@variscout/data/samples`  | Individual sample datasets |
| `@variscout/data/computed` | Pre-computed chart data    |
